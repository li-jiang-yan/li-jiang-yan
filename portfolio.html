JY's Portfolio Developed a web-app that helps job applicants search, filter and sort job openings on LinkedIn. Search results are sorted based on the number of preferred word tokens each job's description contains, helping to save job applicants' energy and time spent from reading job descriptions. Developed a web-app that fetches and word-tokenizes job posts on LinkedIn with a given search criteria. The user can select a few word tokens and the webpage will show a bar chart displaying the number of job posts fetched with the given word tokens. An example use-case would be to quantify the demand of each programming language for a given job search criteria.
