Machine Learning

iris-ml-gravitation-solver (2025)

This is an endeavor to solve the Iris classification machine learning problem adopting ideas of Newtonian gravitational potential. It is able to accurately classify the testing dataset with an accuracy of 97.4%

iris-ml-logistic-regression-solver (2025)

This is a demonstration on solving the Iris classification machine learning problem using a logistic regression solver.

diabetes-ml-linear-regression-solver (2025)

This is a demonstration on predicting the measure of disease progression one year after baseline using linear regression. Here, three different scripts demonstrate the effect of using selections of the features on the regression model's R2 (coefficient of determination) value.

wine-ml-decision-tree-solver (2025)

This is a demonstration on solving the wine classification machine learning problem using decision trees.

wine-ml-logistic-regression-solver (2025)

This is a demonstration on solving the wine classification machine learning problem using logistic regression.

wine-ml-random-forest-solver (2025)

This is a demonstration on solving the wine classification machine learning problem using random forests.
